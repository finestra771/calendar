# 유리 캘린더

## 개발자용 : 빌드

```bash
git clone https://github.com/finestra771/calendar.git
cd calendar

npm install
npm audit fix
```

## 개발자용 : 실행

```bash
npm start
```

## 기술 스택
- React
- html/css/js
- Electron

## 기능
- 달력 보기
- 일정 추가/삭제/수정
- 배너 커스텀
- 이벤트 관리
    - 날짜, 시간 지정 가능, 설명 작성 가능
    - 반복 설정 가능 (매일, 매주, 매월)
    - 일정 자동 넘기기 기능
    - 링크 바로가기 추가 기능
    - 색상 설정 기능
    - 중요 이벤트 지정
- 일기 기능
    - 날짜별로 일기 작성 가능
    - 커스텀 이모지, 폰트, 폰트 크기 지정 가능
    - 일기 목록 보기 및 바로가기
- 빠른 링크 추가 기능
    - 자주 방문하는 웹사이트나 애플리케이션에 대한 바로가기 추가 가능
    - 아이콘과 이름 설정 가능
    - 링크 클릭 시 해당 웹사이트나 애플리케이션이 열림
- 사이드바/스티커 꾸미기 기능
- 커스텀 테마 및 테마 5종
- 디데이 기능
- 유튜브 및 로컬 음악 재생 기능 (임베드 플레이어)
- 이미지 슬라이드쇼
